# Security Policy

## Supported versions

Only the latest release receives security fixes. Update to the most recent
version before reporting an issue.

## Reporting a vulnerability

Do not open a public issue for security vulnerabilities.

Report privately through GitHub Security Advisories:
<https://github.com/mangetoncompost/mirage/security/advisories/new>

If you cannot use that channel, email mangetoncompost@gmail.com with "SECURITY"
in the subject.

Include what you found, how to reproduce it, and the affected version or commit.
Expect an initial response within 7 days. Please give a reasonable window to ship
a fix before any public disclosure.
